# Прогнозирование цены Mercedes

## Модели
- Random Forest: R²=0.73, MAE=24 116
- CatBoost: R²=0.77, MAE=23 048

## Сравнение моделей
| Модель | R² | MAE |
|--------|----|-----|
| CatBoost | 0.77 | 23 048 $ |
| Random Forest | 0.73 | 24 116 $ |

CatBoost показал себя немного лучше.

## Лучшая модель
CatBoost с параметрами: depth=5, learning_rate=0.01, n_estimators=1000

## Важность признаков
Топ-3 фактора, влияющих на цену:
1. Model_GLS (самая важная)
2. Model_S-Class
3. Horsepower

## Вывод
Лучшая модель ошибается в среднем на $23 048. Модель переобучена на конкретные модели авто. Чтобы улучшить модель, нужно либо добавить больше признаков, либо удалить Model и посмотреть, как изменятся метрики.
